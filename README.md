# DeskHop

Windows 11 가상 데스크톱 전환 및 OSD 오버레이 유틸리티

DeskHop은 Windows 11 환경에서 가상 데스크톱 전환을 더 직관적이고 편리하게 만들어주는 가벼운 유틸리티입니다. 단축키를 통한 빠른 전환과 현재 상태를 보여주는 OSD(On-Screen Display) 기능을 제공합니다.

## 🚀 주요 기능

- **가상 데스크톱 단축키 전환**: `Alt + 1` ~ `Alt + 9` 키를 사용하여 지정된 번호의 가상 데스크톱으로 즉시 전환합니다.
- **OSD (On-Screen Display)**: 데스크톱 전환 시 현재 위치를 알려주는 깔끔한 오버레이 창을 표시합니다.
- **모든 데스크톱 가시성**: OSD는 가상 데스크톱을 전환해도 항상 고정된 위치에 나타납니다.
- **자유로운 위치 변경**: 마우스 왼쪽 버튼으로 OSD를 드래그하여 화면 어디든 배치할 수 있습니다.
- **투명도 커스터마이징**: OSD 우클릭 메뉴를 통해 10%부터 100%까지 투명도를 실시간으로 조절할 수 있습니다.
- **시스템 트레이 통합**: 백그라운드에서 조용히 실행되며, 트레이 아이콘의 '닫기' 메뉴를 통해 종료할 수 있습니다.
- **상세 로깅 시스템**: `logs/deskhop.log` 파일에 프로그램의 모든 동작과 에러 상태를 기록하여 문제 발생 시 진단이 용이합니다.

## 🛠 빌드 및 실행 방법

이 프로젝트는 Rust 언어로 개발되었습니다. 빌드를 위해 [Rust 도구 모음](https://rustup.rs/)이 설치되어 있어야 합니다.

```powershell
# 1. 프로젝트 폴더로 이동
cd DeskHop

# 2. 빌드 및 실행
cargo run --release
```

빌드가 완료되면 `target/release/deskhop.exe` 파일이 생성됩니다.

## ⌨️ 사용 가이드

1. **프로그램 시작**: 실행 시 시스템 트레이에 DeskHop 아이콘이 나타나며 백그라운드 모드로 진입합니다.
2. **화면 전환**: `Alt + 1`부터 `Alt + 9`까지의 키를 눌러 원하는 데스크톱으로 이동합니다.
3. **프로그램 이동**: `Ctrl + Alt + 1`부터 `Ctrl + Alt + 9`까지(넘버패드 포함)의 키를 눌러 현재 활성화된 창을 원하는 데스크톱으로 이동시킵니다.
4. **OSD 이동**: 화면에 떠 있는 OSD 창을 마우스 왼쪽 버튼으로 클릭한 채로 드래그하면 위치를 옮길 수 있습니다.
5. **투명도 설정**: OSD 창을 마우스 오른쪽 버튼으로 클릭하여 나타나는 메뉴에서 원하는 투명도를 선택합니다.
6. **프로그램 종료**: 시스템 트레이 아이콘을 우클릭한 후 **'닫기'**를 선택합니다.

## 📂 프로젝트 구조

- `src/main.rs`: 이벤트 루프, 트레이 아이콘, 핫키 관리 등 메인 로직
- `src/osd_window.rs`: GDI 기반의 OSD 윈도우 생성 및 렌더링, 드래그 앤 드롭 구현
- `src/desktop_manager.rs`: Windows 가상 데스크톱 API 연동 레이어
- `src/settings.rs`: 프로그램 설정 관리
- `logs/`: 프로그램 실행 로그가 저장되는 폴더

## 📚 주요 기술 스택

- **언어**: Rust 2021 Edition
- **UI/이벤트**: `winit`, `windows-rs` (GDI)
- **가상 데스크톱**: `winvd` (Windows 11 전용)
- **입력/트레이**: `global-hotkey`, `tray-icon`
- **로깅**: `log`, `simplelog`
